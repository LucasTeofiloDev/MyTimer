# MyTimer
Projeto desenvolvido para praticar lógica de temporização, manipulação avançada de DOM e o conceito de delegação de eventos com JavaScript.

## Funcionalidades
O sistema funciona como um cronômetro em tempo real, permitindo ao usuário iniciar a contagem, pausar o tempo (com feedback visual de cor) e zerar o contador, tudo através de uma interface intuitiva e responsiva.

## Tecnologias
JavaScript (ES6+)

HTML5

CSS3

## Conceitos Aplicados
A implementação foca no uso de Higher-Order Functions como setInterval, manipulação de objetos de data (Date) com formatação internacional (toLocaleTimeString) e Event Delegation, técnica onde um único escutador no document gerencia múltiplos eventos, otimizando o consumo de memória do navegador.
